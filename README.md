# Blogg Spott (Tailwind)

## Overview

This project is a refactored version of the Blog Card design from Frontend Mentor.  
The original version of the project was styled using vanilla CSS. In this lab, the project was refactored to use Tailwindd utility classes instead.

---

## Challenges I Faced

One of the main challenges was learning Tailwind’s spacing scale and understanding how values like `p-4`, `gap-3`, or `mb-5` translate to actual spacing in the layout. I also ran into a few small issues where certain classes were not working because of incorrect syntax or duplicate attributes in my HTML.

Working through those problems required spending time in the Tailwind documentation and experimenting with different utilities until the layout matched the original design.
